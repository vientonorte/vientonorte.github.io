Vientonorte Design System

v1.0.0

Sistema de diseño unificado para todos los proyectos vientonorte

🚀 Getting Started

El Vientonorte Design System proporciona tokens de diseño, componentes y estilos reutilizables para mantener consistencia visual en todos los proyectos.

Instalación vía CDN

Simplemente incluye el CSS en tu HTML:

<link rel="stylesheet" href="https://vientonorte.github.io/cdn/v1/vientonorte.css">

URLs disponibles:

  • https://vientonorte.github.io/cdn/v1/vientonorte.css - Sistema completo
  • https://vientonorte.github.io/cdn/v1/vientonorte.min.css - Versión minificada
  • https://vientonorte.github.io/cdn/v1/tokens.css - Solo tokens
  • https://vientonorte.github.io/cdn/v1/components.css - Solo componentes
  • https://vientonorte.github.io/cdn/v1/icons.svg - Sprite de iconos

📦 Versionado

El design system sigue Semantic Versioning:

  • MAJOR (v2.0.0): Cambios que rompen compatibilidad
  • MINOR (v1.1.0): Nuevas características compatibles
  • PATCH (v1.0.1): Correcciones de bugs

Los archivos publicados en /cdn/v1/ nunca se modifican. Las nuevas versiones se publican en /cdn/v2/, etc.

🤝 Contribuir

  1. Proponer cambios mediante issue en GitHub
  2. Actualizar componentes en /design-system/
  3. Ejecutar ./design-system/build.sh
  4. Validar con projects.json schema
  5. Actualizar CHANGELOG.md
  6. Crear PR con descripción detallada
